# QUARTO - Web Edition

プレミアムなデザインと高度なAI（NPC）を搭載した、ブラウザで遊べるボードゲーム「クアルト」です。
実物のクアルトの思考と緊張感を再現するため、「宣言漏れ指摘」ルールを厳密に管理する独自ロジックを搭載しています。

## 🎮 ゲーム概要
クアルトは、4x4の盤面に16種類の異なる特徴を持つコマを並べていく対戦ゲームです。
特徴的なのは、**「自分が置くコマは、相手が選ぶ」**というルールです。

### 勝利条件
盤面の縦・横・斜めのいずれか一列に、共通の特徴を持つ4つのコマを並べたプレイヤーが「クアルト！」と宣言することで勝利します。

**コマが持つ4つの属性：**
1. **色**: 白 / 黒
2. **形**: 丸 / 四角
3. **高さ**: 高い / 低い
4. **穴**: 有 / 無

### 独自ルール：宣言漏れ
クアルトが成立した瞬間に自分の手番であれば、いつでも「クアルトを宣言！」ボタンを押して勝利できます。
もしクアルトが成立しているのに気づかず次のコマを相手に渡してしまった場合、相手に勝利を奪われる（または宣言権が移る）可能性があります。

## ✨ 特徴
- **プレミアム・デザイン**: ガラスモーフィズムと洗練されたアニメーションを採用。
- **レスポンシブ対応**: PCはもちろん、スマートフォンでも快適にプレイ可能。
- **3段階のNPC**: 完全ランダムなLv.1から、1手先を読んで最適なコマを渡してくるLv.3まで搭載。
- **戦績保存**: ブラウザのLocalStorageを使用して勝敗を記録。

## 🚀 遊び方
1. `index.html` をブラウザで開きます。
2. プレイヤーAが相手（プレイヤーBまたはNPC）に渡すコマを選びます。
3. プレイヤーBがそのコマを盤面に配置します。
4. **自動で手番交代**: 画面が切り替わり、プレイヤーBが新しいコマを選ぶ画面になります。
   - **ポイント**: 相手が置いた瞬間にクアルトが成立していれば、自分の手番ですぐに宣言ボタンを押しましょう！
5. 勝利条件を満たした状態で宣言に成功すれば決着となります。
